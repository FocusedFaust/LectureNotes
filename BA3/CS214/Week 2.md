
### currying, anon functions

### classes

⇒ `val` or `var` in primary constructor make the parameter public
⇒ when no prefix, parameter is public

### auxiliary constructors

⇒ syntax is `def this(param: type) = ...` and to call the constructor: `this(...)`

### string interpolation

⇒ ‘s’ in front of the string quote and ‘$’ in front of the variable name
⇒ arbitrary expressions are written `${...}`
⇒ ‘f’ is an interpolator for formatting, like in Java
